Help & Support

Everything you need to get started with Yappr

play_arrow Quick Start Guide

  1. 1 Get your ElevenLabs API key
    Visit ElevenLabs and create an account to get your API key
  2. 2 Configure the extension
    Click the Yappr icon → Settings → Enter your ElevenLabs API key
  3. 3 Choose your preset
    In the popup, select "Basic", "Email", or "Professional" enhancement preset
  4. 4 Start recording
    Press Cmd+Shift+U (Mac) or Ctrl+Shift+U (Windows) anywhere on the web

auto_fix_high Enhancement Presets

  • Basic: Removes filler words, fixes grammar, cleans transcription artifacts
  • Email: Formats your speech into professional email structure
  • Professional: Transforms casual speech into formal business language

Note: AI enhancement requires an OpenAI API key (~$0.0001-0.0003 per use)

help_outline Troubleshooting

Recording not starting?
  • Check that your ElevenLabs API key is configured in Settings
  • Make sure your microphone permissions are enabled
  • Try clicking the "Start Recording" button in the popup
Text not enhancing?
  • Ensure you've selected a preset (Basic, Email, or Professional)
  • Add your OpenAI API key in Settings for AI enhancement features
  • Check that you have sufficient API credits
Text not inserting into webpage?
  • Click on the text field before recording
  • If insertion fails, text is automatically copied to your clipboard
  • Try pasting with Ctrl+V (Windows) or Cmd+V (Mac)

security Privacy & Security

Yappr is built with privacy as a core principle:

  • ✅ All data stored locally in your browser
  • ✅ API keys never leave your device (except to call the services you choose)
  • ✅ No analytics, tracking, or external servers
  • ✅ Audio only sent to your chosen STT provider (ElevenLabs)
  • ✅ Transcripts only sent to OpenAI if you use AI enhancement
